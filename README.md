TalentRecruit AI 🚀
AI-Enhanced Recruitment & Talent Suite

TalentRecruit AI is a modern, intelligent platform designed to streamline and enhance the recruitment process. It leverages the power of Google's Gemini AI to help hiring teams create, optimize, and manage job descriptions, while providing a clear pipeline view of all active job openings.

✨ Key Features
AI-Powered Job Description Creation: Generate and optimize target job descriptions using Google's Gemini AI, saving valuable time and ensuring consistency.

Active Jobs Pipeline: Get a clear, visual overview of all current job openings, their status (e.g., Full-time, Contract), and locations.

Job Management: Easily create, edit, and manage job postings from a central dashboard.

AI Simulation Mode: (Feature inferred from the UI) Explore and test AI-driven recruitment scenarios to improve your hiring strategy.

🖥️ Local Installation & Setup
Follow these steps to get the project running on your local machine.

Clone the repository

bash
git clone https://github.com/your-username/talentrecruit-ai.git
cd talentrecruit-ai
Install dependencies
For the backend:

bash
cd backend
npm install
For the frontend:

bash
cd ../frontend
npm install
Set up environment variables
Create a .env file in the backend directory. You'll need to add your Gemini API key and database connection string.

env
# Example .env file for backend
PORT=5000
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING
Create a similar .env file in the frontend directory if it requires specific environment variables (e.g., API base URL).

Run the application

Start the backend server:

bash
cd backend
npm run dev
Start the frontend development server:
Open a new terminal window.

bash
cd frontend
npm run dev
Access the application
Once both servers are running, open your browser and navigate to http://localhost:5173 (or the port specified by Vite).

📄 Usage Guide
Creating a Job
Navigate to the "Active Jobs Pipeline" on the dashboard.

Click the "Create" or "New Job" button.

Fill in the job details (Title, Department, Location, Type).

Use the "Optimize with AI" feature to generate a compelling description and key competencies based on your input.

Save the new job posting.

Managing the Pipeline
The dashboard provides a high-level view of all active jobs. You can:

Edit: Update job details or requirements.

View Candidates: (Future feature) Manage applicants for a specific role.

Change Status: Move jobs between stages (e.g., Open, Reviewing, Closed).
